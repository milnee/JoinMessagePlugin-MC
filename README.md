# JoinMessage Minecraft Plugin

## Description
A Minecraft Spigot/Paper plugin that customizes join messages for players, allowing server administrators to personalize player welcome messages.

## Features
- Customizable join messages
- Color support for messages
- Easy configuration

## Installation
1. Download the latest JAR file from the releases section
2. Place the JAR in your server's `plugins` folder
3. Restart the server or reload plugins

## Configuration
Edit the `config.yml` file in the plugin's folder to customize:
- Join message format
- Message colors
- Additional join message settings

### Example Configuration
```yaml
join-message: "&a[+] &f{player} joined the server!"
```

## Requirements
- Minecraft Server running Spigot or Paper
- Java 8 or higher

## Building from Source
1. Clone the repository
2. Ensure you have Maven installed
3. Run `mvn clean package`
